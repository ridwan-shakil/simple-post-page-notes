A lightweight plugin to add private admin-only notes to posts, pages, products and on any custom post types.

== Description ==
Simple Post Page Notes plugin lets adminis to attach short  notes inside the post editor. Notes are stored as post meta and are not visible on the front-end.

== Installation ==
1. Upload the `simple-post-page-notes` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Settings → Post Page Notes to exclude any post types where you don't want notes.

== Frequently Asked Questions ==
= Are notes visible on the front-end? =
No. Notes are stored as post meta and are only shown inside the WordPress admin.

= Can I exclude custom post types? =
Yes. Use Settings → Simple Post Page Notes to choose which public post types to exclude.
